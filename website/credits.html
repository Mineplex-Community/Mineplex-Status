Who made this super sexy site?

This Mineplex status page was created for educational purposes by two morons/high school students over the course of a weekend, looking to expand their knowledge in JavaScript (for the website) and Java (for the backend API) through a simple, but useful project.

Like most of our projects, the Mineplex Status Page is (mostly) open source. Unfortunately, since our API uses private access tokens (and due to a lack of wanting to remove them), it is not publicly visible. However, you can view the source code to this website on GitHub.

If you want to get in contact with us for whatever reason (to report issues, suggest features, or help out), we'd love to have you. Go ahead and send us a message through the Mineplex Forums and we'll get back to you when we can (hopefully).

We hope you enjoy our site,
- Rey & Fallen
